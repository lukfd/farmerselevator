Latest News

Version 1.0.2

Wednesday, March 9, 2022

Connecting farmers with elevators is our goal. There are a lot of tools that will help us reach this goal. With version 1.0.1 we finally have our cloud database so that we can scale our applications even faster. Now with version 1.0.2 we wanted to have a more complete website. We have added:

  • Pagination system in our homepage table
  • Our map will show elevators
  • Save orders' details in a PDF
  • Improved signup process
We are always improving our website and adding more functionality. Our offerings should improve in quality, that's why we are striving to add the possibility to pay directly online for our future release.

Version 0.0.3

Sunday, November 21, 2021

For more than a month we have been working tirelessly on new feature. Our main goal was to reload the home page whenever a new order was sent by a farmer, so that the user would not need to refresh the home page to check if a new order was added to the list. In order to do that, we decided to add flask-socketio to our tools. This new library provided us a new way of sharing information between our application, allowing us to create also a messaging tool so that farmers and elevator will be now able to send private messages live. We now have a more dynamic application that will allow our user to be send information instantly. Here is a recap of the new features:

  • Homepage will now reload whenever a new order is received.
  • Added a new chatting application.
  • After a new order is sent, the user will be redirect to the homepage.

Restyling the application

Tuesday, September 21, 2021

Farmers&Elevators has always the goal of releasing applications that are easy to use and have a good appealing. "Hello web design" by Tracy Obsborn was a great inspiration for us. This amazing book taught us how important are to make our application look modern, therefore we had to redesign our website. We decided to change our font, colorscheme and most importantly we decided to make use of Bootstrap. We are looking forward to improve our website and make it more accessible to all people.
